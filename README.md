# Dopamine Tactics - Perfect Counter v2

이번 버전은 캐시 문제를 피하기 위해 index.html에서 config.js와 script.js에 `?v=perfect2`를 붙였습니다.

## 완벽상성 기준

부분적으로 유리한 캐릭터는 제외하고, 필요한 속성 조합과 정확히 일치하는 캐릭터만 표시합니다.

- 상대 지풍 → 필요 화풍 → 마지/시바
- 상대 지수 → 필요 지풍 → 에이브
- 상대 수화 → 필요 지수 → 크랩
- 상대 화풍 → 필요 수화 → 바트/호머
