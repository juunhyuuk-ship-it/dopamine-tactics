# Dopamine Tactics - Google Sheets 진단 버전

이 버전은 Google Sheets를 먼저 읽고, 실패하면 로컬 data.csv를 읽습니다.
실패 원인은 화면 하단의 진단 내용으로 표시됩니다.

## 반드시 업로드할 파일

- index.html
- style.css
- script.js
- config.js
- data.csv
- README.md

## 확인 주소

업로드 후 아래 주소들을 직접 열어보세요.

- /config.js
- /data.csv
- /script.js

## 완벽 상성 표시 기준

이번 버전은 부분 상성 캐릭터를 제외하고, 상대 속성을 모두 카운터칠 수 있는 캐릭터만 표시합니다.

상성 기준:
- 지는 수에게 강함
- 수는 화에게 강함
- 화는 풍에게 강함
- 풍은 지에게 강함

예시:
- 상대 지풍 → 필요한 우리 속성 풍화 → 화풍 보유 캐릭터만 표시
- 상대 지수 → 필요한 우리 속성 풍지 → 지풍 보유 캐릭터만 표시
- 상대 화풍 → 필요한 우리 속성 수화 → 수화 보유 캐릭터만 표시
- 상대 수화 → 필요한 우리 속성 지수 → 지수 보유 캐릭터만 표시
